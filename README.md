**Molecular Miner** is a high-energy, web-based arcade game designed to help 8th-grade science students master the **Classification of Matter** and the **Law of Conservation of Mass**. 

---

## 🚀 Live Demo
To play the game, simply open the `index.html` file in any modern web browser.

## 🎯 Educational Standards (Texas TEKS)
This game is specifically aligned with the following middle school science standards:
* **TEKS 8.6(A):** Differentiate between elements, compounds, and mixtures, including homogeneous and heterogeneous mixtures.
* **TEKS 8.6(B):** Demonstrate and explain the **Law of Conservation of Mass** using balanced chemical equations.

## 🎮 Game Mechanics
* **Mining:** Use the mechanical claw to grab underground matter.
* **Scanning:** Identify if the captured matter is an **Element**, **Compound**, or **Mixture**.
* **Conservation Check:** Randomly occurring events require players to verify if an equation follows the Law of Conservation of Mass.
* **Boss Level:** Once you reach **500 points**, you must balance a complex equation ($Fe + O_2 \rightarrow Fe_2O_3$) to win!

## 🧪 Scientific Reference Key
| Type | Representation | Scientific Rule |
| :--- | :--- | :--- |
| **Element** | Gold (Au), Silver (Ag) | Pure substance; one type of atom. |
| **Compound** | Water ($H_2O$), Salt ($NaCl$) | Two or more elements chemically bonded. |
| **Mixture** | Brass, Granite | Physically combined; can be separated. |
| **Conservation** | $2H_2 + O_2 \rightarrow 2H_2O$ | Mass in = Mass out (Balanced atoms). |
